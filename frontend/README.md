# IsoTest AI - Frontend

React.js dashboard for test analytics, reporting, and platform management.

## Technology Stack

- React 18
- Vite (Build tool)
- Material-UI (Component library)
- Recharts (Analytics visualization)
- Axios (HTTP client)

## Setup

```bash
cd frontend
npm install
npm run dev
```

Application will be available at: `http://localhost:3000`
